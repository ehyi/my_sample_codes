1. How to change the overdue sound (i.e., Ding)?
Replace wav/notify.wav with the desired wav file.  Make sure  it is named notify.wav

2. Configuration
Run SushiConfig.exe and set all the parameters before using Sushi.exe for the first time.

Parameters:
a. Store Name - The name of the restaturant.
b. Barcode Length - The lenth of the barcode.  All the barcodes used in the restaurant must have the same length.
c. Gaiten Total Minutes - Number of minutes to take for one complete trip of the sushi rail.
d. Plate Maximum Time - Each colored plate's maximum number of minutes allowed on the sushi rail.  Allowed time of the food display.
